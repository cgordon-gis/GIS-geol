Obtaining GIS software

Information correct as of 1st June 2020.

ArcGIS Pro

It is worth reading through all options for ArcGIS Pro before deciding which will work best for you.

Using ArcGIS Pro in University clusters

ArcGIS Pro is available on the desktop on computers in University clusters.

Downloading and installing ArcGIS Pro on your own computer

ArcGIS Pro is only available for Windows computers. ArcGIS Pro is available from the IT Shop for free at:

https://store.leeds.ac.uk/product-catalogue/the-it-shop/software-for-students/mapping-gis/arcgis-pro

Once you have ordered it there will be a delay of a few days before you are sent the download link. You will need to download it from a University machine, or one connected to the University network (e.g. Eduroam), or via Desktop Anywhere. Warning: The download is about 4 GB. You can download the files on one machine, save them on a USB drive, and install them on a different machine if you need to.

You will also need to contact IT to request an ESRI username and password - this is not the same as your University login details.

Using ArcGIS Pro offline

The following information applies if you have no internet access at home, or only limited access. Note that I have not tried this myself so can’t guarantee that it works.

To get an offline license, launch ArcGIS Pro with a network connection and open a new project. Once open, go to the Project tab under Licensing where there is a box to check for “Authorize ArcGIS Pro to work offline.”

Just be aware, once a per-user license goes offline, no more ArcGIS Pro instances can be launched on other machines by that user.

Using ArcGIS Pro on the Student Virtual Windows Desktop

From April 2020 it is now possible to log in to the Student Virtual Desktop to use ArcGIS Pro. The Virtual Desktop will run on Windows, Mac and Linux. You need to install a client on your machine for the best access.

Please make sure that you read all of the information at the link below before you go ahead with this:

https://it.leeds.ac.uk/it?id=kb_article&sysparm_article=KB0014548

The important things to note are that

  • capacity is limited, so if you can install ArcGIS Pro on your own computer using the instructions above, please do so.
  • again, capacity is limited, so if you do access ArcGIS Pro by this route, make sure that you log off the Virtual Desktop when you have finished with it.
  • ArcGIS Pro runs very slowly in the Virtual Desktop if your data is on your M: drive. To get around this, copy your data to the Desktop, work on it, then copy it back to either your M: drive, or OneDrive. Data on the Desktop is deleted when you log out of the Virtual Desktop, or are timed out.
